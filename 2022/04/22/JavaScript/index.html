JavaScript 中的相等性判断

严格相等 ===

如果两个被比较的值具有不同的类型，这两个值是不全等的

如果两个被比较的值类型相同，值也相同，并且都不是 number 类型时，两个值全等

如果两个值都是 number 类型，当两个都不是 NaN，并且数值相同，或是两个值分别为 +0 和 -0 时，两个值被认为是全等的
